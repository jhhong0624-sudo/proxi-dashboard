↑ OTA 펌웨어 업데이트
대상 기기
펌웨어 URL (GitHub Raw 또는 공개 URL)
ESP32가 해당 URL에서 bin 파일을 다운로드하여 자동 업데이트합니다.
GitHub에 bin 파일을 올린 뒤 Raw URL을 입력하세요.
업데이트 완료 후 자동 재부팅됩니다.

GitHub bin 파일 올리는 방법
1. Arduino IDE → 스케치 → 컴파일된 바이너리 내보내기
2. 생성된 .bin 파일을 GitHub 저장소에 업로드
3. 파일 클릭 → Raw 버튼 클릭 → URL 복사
4. 위 입력창에 붙여넣기 후 OTA 시작
⊞ 로그 파일 다운로드
로그 파일은 ESP32에 직접 접속하여 다운로드합니다.
같은 네트워크에 있어야 접근 가능합니다.
◎ Wi-Fi 설정 안내
Wi-Fi 설정은 보안상 원격 변경이 불가능합니다.
아래 절차를 따라 현장에서 직접 설정하세요.
1단계기기 전원 켜기
2단계Wi-Fi: PROXI-XXXX 연결 (PW: 12345678)
3단계브라우저에서 http://192.168.4.1 접속
4단계Wi-Fi 설정 메뉴에서 SSID/PW 입력 후 저장
5단계기기 자동 재부팅 후 MQTT 연결 확인
